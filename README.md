# KODE

## About
KODE is a high-performance collective bridging the gap between academic theory and production-grade engineering. We are more than a club; we are an ecosystem of future-ready architects and engineers dedicated to mastering the art of modern software craftsmanship.

## Features
- **Responsive Design**: Optimized for all devices.
- **Dark Mode**: Integrated theme toggle for comfortable viewing.
- **Modern UI**: Utilizing Google Fonts (Outfit) and Remix Icons.
- **Interactive Elements**: Smooth transitions and meaningful animations.

## Technologies Used
- **HTML5**: Semantic markup for structure.
- **CSS3**: Custom styling and layout logic.
- **JavaScript**: Interactive functionality and DOM manipulation.

## Project Structure
```
/
├── index.html        # Main landing page
├── pages/            # Inner pages (About, Events, Gallery, Contact)
├── css/              # Stylesheets
├── js/               # Scripts
└── assets/           # Images and media files
```

## Getting Started
1. Clone the repository.
2. Open `index.html` in your preferred web browser.
